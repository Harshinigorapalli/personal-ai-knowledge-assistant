# 📄 Personal AI Knowledge Assistant

![Demo](demo.gif)

An AI-powered app that allows users to upload PDFs and ask questions based on document content using semantic search.

## Features
- 📂 Upload PDF
- 🤖 Ask questions
- ⚡ Fast semantic search (FAISS)
- 💬 Chat-like interface

## Tech Stack
- Python
- Streamlit
- Sentence Transformers
- FAISS
